🚀 Bale Web Chat Scraper | اسکرپر و پشتیبان‌گیر پیام‌رسان بله

یک ابزار سریع و کاربردی برای استخراج، دانلود و پشتیبان‌گیری از تاریخچه چت‌ها در نسخه وب پیام‌رسان بله (**Bale Messenger**). این برنامه پیام‌ها را با ترتیب کاملاً خطی، مرتب و بدون به‌هم‌ریختگی با نمایش زمان ارسال پیام استخراج کرده و به صورت فایل اکسل تحویل می‌دهد.

---

## 📦 دانلود نسخه آماده (بدون نیاز به نصب پایتون - مخصوص عموم)

اگر با برنامه‌نویسی آشنایی ندارید یا پایتون روی سیستم شما نصب نیست، اصلاً نگران نباشید! کافیست مراحل زیر را بروید:

۱. از لیست فایل‌های بالای همین صفحه، فایل **`release.zip`** را دانلود کنید و از حالت زیپ خارج (Extract) کنید.

۳. فایل اجرایی با پسوند `.exe` را باز کرده و طبق دستورالعمل‌های ساده‌ای که روی صفحه ترمینال نشان داده می‌شود پیش بروید.

---

## 🛠️ نحوه نصب و اجرا از سورس‌کد (مخصوص برنامه‌نویسان)

اگر توسعه‌دهنده هستید و می‌خواهید کدهای پایتون پروژه را روی سیستم خود اجرا، تست یا ویرایش کنید، مراحل زیر را دنبال کنید:

### ۱. نصب پیش‌نیازها
ابتدا کتابخانه‌های مورد نیاز را از طریق فایل `requirements.txt` نصب کنید:
```bash
pip install -r requirements.txt

```

### ۲. نصب مرورگر Playwright

برای دانلود و فعال‌سازی درایور مرورگر پایتون، دستور زیر را در ترمینال بزنید:

```bash
playwright install

```

### ۳. نحوه استفاده از اسکریپت‌ها

* **قدم اول - آنالیز ساختار چت (اختیاری):**
اگر بله به تازگی آپدیت شده و کلاس‌های HTML آن تغییر کرده است، ابتدا فایل `scout.py` را اجرا کنید تا ساختار جدید کلاس‌ها را در فایل `bale_structure_report.txt` به شما تحویل دهد.
```bash
python scout.py

```


* **قدم دوم - شروع استخراج واقعی:**
فایل اصلی پروژه را اجرا کنید:
```bash
python main.py

```


مرورگر باز می‌شود. وارد حساب بله خود شوید، چت مورد نظر را باز کنید و صفحه را به نقطه‌ای که می‌خواهید استخراج از آنجا شروع شود (مثلاً یک تاریخ خاص در گذشته) ببرید. سپس در ترمینال کلید **Enter** را بزنید تا فرآیند اسکرول اتوماتیک و ذخیره‌سازی شروع شود.

## ✨ ویژگی‌های برجسته (Features)

* **حفظ ۱۰۰٪ ترتیب خطی:** استخراج پیام‌ها دقیقاً به همان ترتیبی که ارسال شده‌اند (بدون مشکل سورت کاتوره‌ای مرورگر).
* **پشتیبان‌گیری هوشمند (Chunking):** ذخیره خودکار پیام‌ها در پارت‌های ۵۰۰ تایی به صورت فایل‌های تمیز `CSV` (قابل خروجی گرفتن و باز کردن در اکسل).
* **نام‌گذاری خودکار فایل‌ها:** تشخیص هوشمند شماره پارت بعدی و درج تاریخ و ساعت شروع هر پارت در نام فایل خروجی.
* **ذخیره سشن مرورگر:** استفاده از مرورگر پایدار (`Persistent Context`)؛ یعنی فقط یک‌بار لاگین می‌کنید و در دفعات بعدی نیازی به اسکن مجدد QR کد یا دریافت کد تایید نیست.
* **اسکریپت آنالیزور ساختار (Scout):** دارای یک ابزار اختصاصی برای استخراج سریع کلاس‌های HTML بله، جهت عیب‌یابی و آپدیت آسان کدهای اصلی در صورت تغییرات فرانت‌اند بله.
* **بهینه‌سازی برای اکسل:** ماسک کردن تاریخ‌های تکراری و پشت‌سرهم برای خوانایی و زیبایی بیشتر فایل خروجی.

## ⚠️ نکات امنیتی و حریم خصوصی

* پوشه `bale_user_data` که پس از اولین اجرا ساخته می‌شود، حاوی سشن و اطلاعات ورود شما به پیام‌رسان بله است. **هرگز** این پوشه را در فضای مجازی یا گیت‌هاب آپلود نکنید (این پوشه به طور پیش‌فرض در فایل `.gitignore` قرار داده شده است تا آپلود نشود).
* تمام خروجی چت‌های شما در پوشه `bale_backups` و کاملاً آفلاین روی سیستم خودتان ذخیره می‌شود و هیچ داده‌ای به سرور خارجی ارسال نمی‌گردد.
